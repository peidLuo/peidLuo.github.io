Selenium与自动化测试

发表于 2017-07-23 | 分类于 测试

Selenium是最流行的浏览器自动化测试工具。最易使用的方法是使用基于Selenium的WebDriver API，它通过调用浏览器接口实现自动化，执行诸如“打开网页”、“移动网页上的元素”、“点击链接”、“查看链接是否打开URL”等。对于运行自动化测试来说是一个十分理想的方法。
